## Video #1
- A graph is an ordered pair with vertex and edge sets, they can be drawn differently
- Vertices = Nodes
- Complete graph: All vertices connect via all edges defined in the set
- Simple graph: Unidirectional graph where 
    - vertices have only one edge connection
- Equality of graphs: Vertex and edge sets are the same
- Empty graph: vertices exist but no empty edge set
- Null graph: empty graph with 0 vertices
- Graphs represent objects and their relations
